## 依赖说明
依赖对应版本的elasticsearch

## 开启插件

```js
// config/plugin.js
exports.elasticsearchClient = {
  enable: true,
  package: 'egg-elasticsearch-client',
};
```

## 使用场景

- 这是一个es客户端插件，插件库的版本与es的主版本对应

## 详细配置

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。
